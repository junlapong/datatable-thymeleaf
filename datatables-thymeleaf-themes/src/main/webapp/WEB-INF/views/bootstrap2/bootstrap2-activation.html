Bootstrap 2 theme

This example shows how easily the Bootstrap v2 framework can be integrated.

Note that we made the black row in the thead disappear. To do that, we used the Dandelion Thymeleaf dialect to modify the active scopes by excluding the Datatables' CSS as follows:

ddl:css-excludes="datatables"

As a consequence, we lost the sorting icons. But you could of course get them back by using custom CSS. See the documentation for more details.
